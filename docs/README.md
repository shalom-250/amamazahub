## 🌍 About AmazamaHub

AmazamaHub is a TikTok-style short video platform focused on:
- 🇷🇼 **Rwanda-first** community and local discovery
- 🎬 **Short-form video** creation, sharing, and interaction
- 📍 **Location-aware** feed personalization
- 🔴 **Live streaming** with real-time chat
- 🛒 **Integrated shop** for creators to sell products
